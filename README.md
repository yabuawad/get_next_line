This project is about using system calls to read from a file or even the terminal line by line with a dynamically allocated buffer which taught me memory mangement -the hard way lol-

 ~What it does :
- Reads from any file descriptor one line at a time    
- Manages memory carefully to avoid leaks  
- Returns `NULL` when there’s nothing left to read
